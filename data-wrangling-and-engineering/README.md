# Data Cleaning & Feature Engineering Pipeline

## Overview
A comprehensive workflow for transforming raw, noisy data into a model-ready dataset.

## Core Processes
* **Data Cleaning:** Cleans corrupted instances and handles noisy placeholders (e.g., '?' → `NaN`).
* **Imputation:** Robust strategies for handling missing values to ensure integrity.
* **Feature Engineering:** Transforms raw inputs into optimized numerical formats via encoding and scaling.
* **Schema Consistency:** Enforces strict alignment between training and validation sets to prevent pipeline errors.

---
**Technical Summary:** A comprehensive data preparation notebook focused strictly on cleaning, imputing missing values, and engineering features to construct a model-ready dataset.
