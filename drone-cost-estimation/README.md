# Drone Cost Estimation & Classification

## Overview
This project explores hardware cost modeling through a dual-modeling approach, transitioning from continuous cost estimation to binary classification.

## Technical Approach
* **Regression Analysis:** Modeled continuous cost estimation to understand underlying financial drivers.
* **Binary Classification:** Binarized the target variable to architect a Support Vector Classifier (SVC) using an RBF kernel.
* **Performance Evaluation:** Validated model effectiveness using a confusion matrix to analyze precision and recall.

---
**Summary:** Predictive Cost Modeling & Classification: Developed dual predictive models for hardware costs. Explored continuous cost estimation (Regression), then binarized the target variable to architect a classifier (SVC with RBF kernel), and evaluated performance via a confusion matrix.
