# Incarceration Trend Analysis

## Overview
This project uses demographic and behavioral data to predict classification trends. I focused on building automated, reproducible machine learning pipelines.

## Key Technical Achievements
* **Automated Pipelines:** Used `ColumnTransformer` to handle imputation, scaling, and encoding for mixed data types.
* **Ensemble Modeling:** Trained a `RandomForestClassifier` to predict complex categorical targets.
* **Robust Evaluation:** Used $k$-fold cross-validation and Macro F1-scores to ensure the model remains accurate despite class imbalances.

---
**Summary:** Demographic & Behavioral Classification: A complex societal dataset analyzed by architecting automated preprocessing pipelines, training ensemble models, and validating results via cross-validation and F1-scores.
